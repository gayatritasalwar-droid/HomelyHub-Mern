# HomelyHub – Property Booking Web Application

# About
HomelyHub is a full stack property booking web application developed using the MERN stack during my internship.
The application allows users to view, rent, or list properties for sale and explore available properties based on location.

# Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

# Features
- Property listing with details
- Rent and sale property options
- Area-wise property browsing
- User-friendly interface
- Backend API integration
- Basic CRUD operations
